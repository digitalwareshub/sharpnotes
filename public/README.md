# Images Directory

## Profile Image

Place your profile image here as `kam.jpg` (or `kam.png`)

**Recommended specs:**
- Format: JPG or PNG
- Dimensions: 256x256px (square)
- File size: < 100KB
- The image will be displayed as a small 64x64px avatar

The image is configured to:
- Display at 64x64px (small avatar size)
- Be non-clickable (`pointer-events-none`)
- Be non-draggable (`draggable="false"`)
- Be non-selectable (`select-none`)
- Circular cropping with rounded-full
- Cover the entire circle area (`object-cover`)
